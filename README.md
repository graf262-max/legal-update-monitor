# 법률 업데이트 모니터링 시스템

한국 주요 법률의 제·개정 사항을 매일 자동 모니터링하는 시스템입니다.

## 🔍 모니터링 대상
- 상법, 민법, 개인정보보호법, 직업안정법
- 정보통신망법, 전자금융거래법, 채용절차법
- 약관규제법, 공정거래법, 저작권법

## 📡 데이터 소스 (7개 기관)
- 국가법령정보센터 (API)
- 열린국회정보 (API)
- 고용노동부 (RSS)
- 개인정보보호위원회, 과기정통부, 금융위, 공정위 (스크래핑)

## 🚀 배포
Netlify에서 매일 09:00 KST에 자동 실행됩니다.

## ⚙️ 설정
1. `.env.example`을 `.env`로 복사
2. API 키 입력 (선택사항)
3. `npm install` 후 `npm run test:collectors`로 테스트
